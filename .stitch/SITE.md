# Aster Perp Trading Dashboard - Site Vision

## 1. Vision
A premium, "analyst-grade" trading dashboard for Aster Finance. The UI should feel sophisticated, data-dense but clean, using high-contrast dark mode aesthetics with subtle animations. It should look like a professional trading platform (e.g., Bloomberg, Binance Futures, TradingView).

## 2. Target Audience
Active perpetual contract traders who need real-time P&L monitoring, strategy execution control, and historical trade analysis.

## 3. Core Features
- Real-time P&L and account equity tracking.
- Live position management.
- Dynamic strategy controls (Enable/Disable/Params).
- Detailed trade history.
