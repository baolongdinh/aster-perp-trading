# Spot WebSocket v1 – Reference

**Market payloads:** aggTrade (a, p, q, f, l, T, m); trade (t, p, q, T, m); kline (k: t, T, o, c, h, l, v, n, x, q); depthUpdate (e, U, u, pu, bids, asks or b, a); miniTicker/ticker (s, c, o, h, l, v, q, …); bookTicker (u, s, b, B, a, A).

**User stream:** outboundAccountPosition (B: [{a, f, l}], e, T, E, m). executionReport (s, c, S, o, f, q, p, ap, P, x, X, i, l, z, L, n, N, T, t, m, O, Z, Y, Q, e, E).
